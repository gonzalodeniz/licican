# Epicas de Producto

## Objetivo
Separar la especificacion funcional consolidada en artefactos por epica para facilitar trazabilidad, refinamiento y futura asignacion de trabajo.

## Epicas disponibles
- `e-01-descubrimiento-y-relevancia.md`
- `e-02-ingestion-y-trazabilidad-del-dato-oficial.md`
- `e-03-retencion-y-seguimiento-operativo.md`
- `e-04-experiencia-transversal-y-gobierno.md`

## Regla de mantenimiento
- Cada fichero de epica debe mantenerse sincronizado con `product-backlog.md`, `historias-de-usuario.md`, `casos-de-uso.md`, `roadmap.md` y `especificacion-funcional-gestion-licitaciones.md`.
